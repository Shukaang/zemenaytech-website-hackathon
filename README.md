## Quick Start

### Prerequisites

- Node.js 18+
- npm or yarn
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/Shukaang/zemenaytech-website-hackathon.git

# Navigate to project directory
cd zemenaytech-website-hackathon

# Install dependencies
npm install

# Start development server
npm run dev
```
